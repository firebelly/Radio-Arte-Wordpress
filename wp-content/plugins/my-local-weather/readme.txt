This plugin will display the local weather conditions from Yahoo! Weather.



== Description ==



Now you can easily display the current local weather in your area on your website.  This plugin will pull the current weather forecast directly from Yahoo! Weather to give you the latest up to date weather conditions.



-RECENTLY ADDED: Sidebar Widget.

-Server compatibility by utilizing WordPress' built in tools.





== Installation ==



1. Download the plugin, extract and upload it to your plugins folder on your server.



2. Activate the plugin.



3. Go to Settings (Options in other WP versions), and Local Weather.



4. Enter your area's zip code and hit "Save".



5. Add this code to your theme where you want the weather to be displayed: 



&lt;?php if (function_exists(my_local_weather)) {  my_local_weather(); } ?&gt;	





Note: The link back is not required, but it would be nice to have it... :-)
